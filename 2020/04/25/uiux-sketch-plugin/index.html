推薦的Sketch Plugin

Posted on 2020-04-25 | In UI/UX |

紀錄好用的Sketch Plugin。

  1. sketch-measure
  2. sketch-iconfont
  3. Sketch-SF-UI-Font-Fixer
  4. Anima
  5. Artboard-manager
  6. Craft

Sketch-measure

GitHub - utom/sketch-measure
用來測量圖層的長寬，可用來打草稿，工程師接手後也能清楚知道寬距、間距要怎麼設置。

Sketch-iconfont

GitHub - keremciu/sketch-iconfont
管理多種icon字型，可以快速搜尋和插入。

Sketch-SF-UI-Font-Fixer

GitHub - puretears/Sketch-SF-UI-Font-Fixer
可以調整字體的間距，這是根據Apple iOS人機手冊(Typography - Visual Design - iOS - Human Interface Guidelines - Apple Developer)中所提出的字體在不同大小需要擁有不同的間距，同樣的在Sketch設計中也需要符合這樣的條件，設計跟開發出來才會有一樣的效果。
原作者已經停止更新，這裡下載的是泊學作者的版本，已經更新到iOS13的版本，另外，他們的影片真的很優秀，真心推薦😀

Anima

Anima - Design to code, automated.
可以在Sketch中做出跟auto layout相同的效果，不僅在設計中方便，在變換不同Artboard的Driver也能適應。

Artboard-manager

GitHub - bomberstudios/artboard-manager
便利且迅速排版多個Artboard

Craft

Craft | InVision
它的功能實在太強大了，我目前只拿來管理字型、icon、顏色等等，可以儲存在Cloud上，其他專案也方便使用，其他等有用到在做分享。
