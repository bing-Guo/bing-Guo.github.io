Google Map API練習

Posted on 2020-05-04 | In iOS開發 |

本來要去面試一間公司，他們的APP是有使用到Google Map的，就花了兩天練習，雖然最後因為一些原因沒面試，這就來稍稍紀錄一下。

安裝與API Key

  • 官方文件：Get an API Key
  • 官方文件：Get Started
  1. 你得先取的API Key，這照著上面文件做應該沒問題
  2. Cocoapod安裝GoogleMaps跟GooglePlaces
  3. 在Info.plist定義URL schemes，這可以讓你在APP中呼叫啟動Google map APP
  4. 接著我會在AppDelegate.swift的didFinishLaunchingWithOptions方法中，加入API Key，大概會像下面這樣，這部分我有特別將API驗證過程封裝到GoogleMapInitializer.swift，在AppDelegate.swift呼叫

GMSMapView

  • Map Objects

GMSMapView是地圖的本身，官方範例是使用純code去建立，如果是用storyboard，首先要先拉一個UIViewController，再把它的class設為MapViewController，再拉一個UIView，class設定為GMSMapView(以下稱mapView)。

我的程式中是在主畫面以ContainerView去建立，因為預期會在地圖上疊一些圖層，比如說搜尋欄(雖然後來沒有實現搜尋功能 😅

中心點

設定地圖中心點需要先建立一個GMSCameraPosition，可以想像是一個鏡頭(你現在看地圖的範圍的中心點)，建立方式如下方程式，設定經度、緯度以及縮大大小(數字越小地圖越廣)，再將camera設定到一個GMSMapView物件中，地圖就會呈現中心點了。

移動特效

我在取得到用戶的Location後，會將地圖中心點從預設地點轉移到用戶位置，這時會將新的Location轉換成GMSCameraPosition物件，再呼叫mapView.animate(to: camera!)

GMSMarker

  • Markers
    GMSMarker是地圖上的大頭針，我將其建立的過程封裝到GMSMarker當中
  1. latitude和longitude就是經緯度
  2. title跟snippet是Window Info上的資訊(點擊大頭針會有一個小彈跳視窗)，我這裡沒寫snippet是因為只有兩個都非nil才會有Window Info，少一個都不行，因為我不想要所以就拿掉了snippet，title則是拿來存放資料的唯一值，做來點擊大頭針換頁取得更多資訊用。
  3. iconView可以更換大頭針的圖片
  4. tracksViewChanges根據官方說法，如果有重繪Marker需求在設為true，否則re-rendered可能會消耗電池使用。
  5. 這裡選擇將GMSMapView注入，直接Marker標記在map上 值得注意的是GMSMarker在建立時一定要是在main-thread，不然會出錯，所以我選擇在MapViewController中，接收到ViewModel資料更新後，進行Update。

點擊事件

我的範例邏輯是這樣，點擊大頭針，會present到詳細頁面，所以我需要將GMSMarker的title資訊去取得到Pin(我自己設定的Mode，有其他停車場的資訊)，再將詳細頁面的ViewModel設定完成，資料都完成後再進行換頁。

切記要先設置delegate，像這樣mapView.delegate = self

MyLocation & MyLocationButton

MyLocation就是你人現在的位置(藍色點點)，MyLocationButton則是點擊後會focus到你現在位置，這兩個設置很簡單，兩行程式結束

連接到Google Map APP

我將下方程式綁在按鈕事件當中，當點擊按鈕後，取得選定的大頭針位置，將經緯度帶入URL當中，以comgooglemaps://作為開啟Google Map APP的URL，這點實作撥打電話功能的方式一樣，不過記得要先將Info.plist定義好URL schema，這請參照此篇的第一步驟。

後語

以現在收費方式來看，Maps SDK for iOS是免費且無限制，所以一般地圖呈現是免費的，但其他像是導航、景點、街景這些是要收費的，不過下次要做Map我可能會選用Apple提供的地圖，一來看起來導航什麼是免費的，看Youbike APP做起來效果也不錯，二來Xcode內建就有，也許整合性會好多(猜測，我也還沒用過)，以上，謝謝觀賞(倉促結束…🤷‍♂️
